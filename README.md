Este proyecto simula un panel de control **HMI/SCADA** para una planta termoeléctrica que utiliza Metano. Implementa una **Red Neuronal (MLPRegressor)** para predecir comportamientos termodinámicos de gases reales y calcular la eficiencia de conversión de energía en tiempo real.

## 🚀 Características
- **Gemelo Digital:** IA que predice la temperatura basada en presión y volumen (Ecuación de Van der Waals).
- **Cálculos de Energía:** Estimación de Entalpía, Flujo Másico y Potencia en Megavatios (MW).
- **Diagnóstico Industrial:** Monitoreo de latencia de la IA y factor de compresibilidad (Z).

## 🛠️ Tecnologías
- **Python 3.x**
- **Scikit-Learn** (Redes Neuronales)
